# pHAT REST

_Extremely thin HTTP wrapper around the [MicroDot pHAT](https://shop.pimoroni.com/products/microdot-phat)_

## Running it

    make install

then

    python webserver.py

ought to do it. You'll also need `foreman` to install the `systemd` startup scripts, probably

    gem install foreman

then

    make foreman

## API

### `/lights`
